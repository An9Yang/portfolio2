# Martin.R Portfolio 项目架构概览

## 项目结构

### 文档文件
- `docs/prd.md`: 产品需求文档，定义项目目标和设计要求
- `docs/todo.md`: 开发任务清单，跟踪实现进度
- `docs/overview.md`: 项目架构文档（本文件）

### 核心页面
- `src/pages/home.tsx`: 主页面，完全还原Martin Robart个人品牌设计，包含精确的排版层次和布局定位

### 样式系统
- `src/styles/globals.css`: 全局CSS样式定义，包含Martin.R专用的字体层次系统和Inter字体导入
- `tailwind.config.ts`: Tailwind配置，包含温暖棕色色彩系统和自定义字体设置

## 设计实现细节
- **色彩系统**: HSL(28, 20%, 52%) 温暖棕色背景，完美匹配原设计
- **字体层次**: 5级字体大小系统，从intro文字到hero大标题的完整视觉层次
- **布局定位**: 三栏布局（左侧姓名、中央肖像、右侧信息）+ 底部大标题
- **响应式设计**: 桌面端三栏布局，移动端垂直堆叠适配
- **视觉细节**: 右下角装饰圆点、精确的文字间距和行高设置

## 核心特性
1. **100%设计还原**: 精确匹配原始设计的每个视觉元素
2. **排版驱动**: 通过字体大小和位置创造视觉冲击力
3. **专业品牌**: 突出Martin Robart作为数字设计师的专业形象
4. **现代技术栈**: React + TypeScript + Tailwind CSS 实现

## 最终实现细节
- **导航系统**: 左上角Martin.R品牌标识 + @2025年份标记 + 右上角服务菜单
- **主标题区**: "Hey there, I'm Martin Robart" 采用大字号层次展示
- **人物肖像区**: 无容器的图片上传区域，支持透明PNG图片直接显示
- **信息区域**: 右侧垂直分隔线设计，展示地理位置和职业信息
- **底部标题**: "Digital Designer" 超大字号，营造视觉冲击力
- **About Me区域**: 左侧粘性垂直标题 + 右侧个人介绍和数据统计
- **Selected Projects区域**: 左侧粘性垂直标题 + 右侧项目展示
- **粘性滚动效果**: 左侧标题在对应内容区域滚动时保持固定位置
- **装饰元素**: 右下角精致的圆点装饰
- **交互功能**: 点击中央区域可上传人像图片，支持透明背景
- **响应式**: 完整的移动端适配，保持设计完整性
